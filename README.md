## PC Assemble Lab
**Hands-on activity assembling and understanding computer hardware components.**
![4393](https://github.com/user-attachments/assets/4bb9ece9-3b7a-4743-bdc4-df415f193cc2)

**What I have gained:**
I've only had a brief knowledge about PC assembling before attending this session, and definitely I have never had the opportunity to assemble the main parts hands-on. But after attending this session, I've now gained a thorough understanding about the PC assembling process in more detail. My group faced a bit of a problem while we were attempting to run the PC upon building it from scratch. Thankfully, it managed to turn on smoothly afterwards. 

**Suggest any improvement:**
To improve the PC Assemble Lab session, we should focus on having more time to learn each parts of the PC in more detail hands-on. Hence, it will be more interesting and engaging for students.

**Others:**
This experience could be more engaging by introducing a new challenge where teams that complete early are tasked with an extra project. For example, they could take have us disassemble another set of newer PCs for each team and assemble them back again so we can analyze and compare the difference between old and new PCs. Not only this challenges the students more, but it could help pique the students' interest in this matter.
